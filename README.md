Habit Tracker API
Описание
RESTful API для управления полезными привычками и целями с полным набором CRUD операций и статистикой.

Эндпоинты покрывают:

• Управление привычками
• Управление целями
• Отслеживание выполнения привычек
• Получение статистики
Эндпоинты
• Привычки (/api/v1/habits)
• Получить все привычки

Метод: GET
• URL: /api/v1/habits
• Описание: Получить список всех привычек
• Ответ: 200 OK - Массив объектов привычек
• Получить привычку по ID

Метод: GET
• URL: /api/v1/habits/:id
Параметры:
id (в пути) - ID привычки
Ответы:
• 200 OK - Объект привычки
• 404 Not Found - Привычка не найдена
• Создать привычку

Метод: POST
• URL: /api/v1/habits
• Тело: Объект привычки (JSON)
Пример:
{
"name": "Чтение книги",
"description": "Читать 30 минут в день",
"category": "обучение",
"frequency": "ежедневно"
}
Ответы:
• 201 Created - Созданная привычка
• 400 Bad Request - Неверные данные

Обновить привычку
• Метод: PUT
• URL: /api/v1/habits/:id
Параметры:
• id (в пути) - ID привычки для обновления
• Тело: Обновленный объект привычки (JSON)
Ответы:
• 200 OK - Обновленная привычка
• 404 Not Found - Привычка не найдена
• 400 Bad Request - Неверные данные
Удалить привычку

Метод: DELETE
• URL: /api/v1/habits/:id
Параметры:
id (в пути) - ID привычки для удаления
Ответы:
• 204 No Content - Успешно удален
• 404 Not Found - Привычка не найдена
Отметить привычку как выполненную

• Метод: PUT
• URL: /api/v1/habits/:id/complete
Параметры:
id (в пути) - ID привычки
Ответы:
• 200 OK - Успешно отмечено
• 404 Not Found - Привычка не найдена

Статистика
Получить статистику

Метод: GET
• URL: /api/v1/statistics
• Описание: Получить сводную статистику по привычкам и целям
• Ответ: 200 OK - Объект со статистикой
Примеры использования

# Получить все привычки

curl -X GET http://localhost:3000/api/v1/habits

# Создать привычку

curl -X POST http://localhost:3000/api/v1/habits \
 -H "Content-Type: application/json" \
 -d '{
"name": "Пить воду",
"description": "Пить 2 литра воды в день",
"category": "здоровье",
"frequency": "ежедневно"
}'

# Отметить привычку как выполненную

curl -X PUT http://localhost:3000/api/v1/habits/4/complete

# Получить статистику

curl -X GET http://localhost:3000/api/v1/statistics
